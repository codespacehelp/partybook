# partybook

## Installation

Make sure you have [node.js](https://nodejs.org/en) installed.

Open the code in VS Code, then open two terminals:

## Terminal 1

```
$Env:PATH = "C:\node"
npm install
npm run dev
```

## Terminal 2

```
$Env:PATH = "C:\node"
npm run partykit-dev
```

Open the page at http://127.0.0.1:3000/
